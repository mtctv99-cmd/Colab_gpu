# Kế hoạch Đánh giá Server (Server Review Plan)

## Mục tiêu
Đánh giá toàn diện kiến trúc, bảo mật, hiệu năng và các lỗi tiềm ẩn trong hệ thống server FastAPI + Colab Daemon.

## Các file quan trọng cần kiểm tra (Critical Files)
- `/media/reup/Data_sv4/SVRV/Colab/app/main.py`: Entrypoint, middleware và cấu hình CORS/Rate Limit.
- `/media/reup/Data_sv4/SVRV/Colab/app/database.py`: Cấu hình SQLite và Connection Pool.
- `/media/reup/Data_sv4/SVRV/Colab/satellite_node/daemon.py`: Logic của Node Daemon trên VPS phụ.
- `/media/reup/Data_sv4/SVRV/Colab/colab/worker.py`: Script chạy trên Google Colab runtime.

## Các phần sẽ được phân tích sâu
1. **Bảo mật**: Các khóa bí mật (Secret Keys), xác thực WebSocket, kiểm tra quyền truy cập API.
2. **Cơ sở dữ liệu (SQLite)**: Tối ưu hóa SQLite PRAGMA (WAL mode) để tránh nghẽn khi ghi đồng thời (locking).
3. **Đồng bộ hóa mã nguồn**: Giải quyết sự trùng lặp mã giữa `colab/worker.py`, `satellite_node/daemon.py` và `npm-package/core/`.
4. **Xử lý bất đồng bộ & Race Condition**: Race condition khi tải file audio hoàn thành qua HTTP và thông báo qua WebSocket cùng lúc.
